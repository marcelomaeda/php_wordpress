== Installation ==

Installing this plugin as regular wordpress plugin. First you need to install WPBakery Page Builder formerly Visual Composer plugin then install Amazing Hover Effects for WPBakery Page Builder.

This addon plugin works like other page builder addon plugin. Go to new page then click on WPBakery Page Builder builder then click on Add Element you should see Amazing Hover Effects menu as Name "Amazing Hover" on Menu Top, click on it you will see Hover Effects then click on it and browse an image and customize according to your needs.
