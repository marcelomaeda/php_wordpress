=== Description ===

Want to display more attractive client testimonials to your web page ? Advance Testimonial addon for WPBakery Builder (formerly Visual Composer) is perfect solution for you. It's  have 14+ stunning styles that you help for everyday use in website page building.


### View Demo

`
	http://demo.thecodude.com/testimonial-vc-addon/
`

### Features of this Plugin
* 14+ Unique Styles
* Security ensured, use it hassle free & without risk.
* Fully responsive and mobile friendly.
* Adding Testimonials to your current website
* Beautiful, minimalist & light-weight.
* NO coding knowledge required! easy to use.
* Support all modern browsers: Firefox, Chrome, IE, Safari etc.
* Schema.org compliant JSON-LD markup & It will optimized for speed.
* Compatible with any kind of theme.
* More style are coming soon...

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
