== Description ==

This plugin integrates wordpress registration sytem with Interspire email marketing software seemlessly.
Registered users can automatically be subscribed in interspire mailing the wordpress Interspire Users Subscription plugin by [Ajoft Technologies](http://www.ajoft.com/).
Once the plugin is activated all the users who registers on a wordpress site automatically gets subscribed in their mailing list using XML API.

== Installation ==

1. Upload the `interspire` folder to the `/wp-content/plugins/` directory
1. Activate the Interspire User Subscription plugin through the 'Plugins' menu in WordPress
1. Configure the plugin by going to the `Interspire User Subscription` menu that appears in your settings menu
